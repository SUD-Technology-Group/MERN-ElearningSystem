Các chức năng phát triển:
 1. Lưu trữ bài thi ( database )
 2. Đăng nhập / tạo tài khoản ( truyền thống )
 3. Bảo mật đề thi ( Sử dụng twilio send SMS OTP đến phone của tác giả đề thi )
 4. Xác thực tài khoản: ( 2 options )
    - Sử dụng Register Code được cung cấp để tạo tài khoản
    - Sử dụng Excel/Sheet để cho giáo viên điền form và cắt chuỗi auto generate tài khoản 
    ( Nếu là sheet thì mẫu access vào 1 sheet ở link này https://github.com/keysKuo/Nodejs-HHFurniture/blob/main/src/resources/routes/Admin.js)
 5. Triển khai làm bài thi thử và tự chấm điểm


Các UI cần cung cấp:
 1. Các trang đăng nhập / tạo tài khoản
 2. Trang làm bài thi
 3. Trang admin quản lý học sinh, đề thì, ....
